# Deploy
## test
```shell
./dotter deploy test
```

## deploy
```shell
./dotter --help
./dotter help deploy
./dotter deploy
# set proxy url
./dotter deploy --url "https://proxy.xxx.xx/https://github.com/Sakura1943/dotfiles.git"
```
