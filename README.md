# dotfiles

# Deploy
## test
```shell
./dotter deploy test
```

## deploy
```shell
./dotter --help
./dotter help deploy
./dotter deploy
# custom git repository url
./dotter deploy --url "https://github.com/username/dotfiles.git"
```
